Adding users to your institution's account

netRink and its products are designed to be user friendly, and easy to use. Adding users to your institution's account is just like adding friends on any social media platform.

Accounts come with a default free pricing plan which allows only an individual user per account. To upgrade your plan, go to Settings Module -> Subscription. After upgrading your plan, go to Admin Module -> New Invitation. From there it is just the usual modern social media affairs ðŸ˜„.

On the user's side, the user has to have a netRink Personal account to access an institution's account. After signing up, go to Settings Module -> All Accounts -> Invitations. Accept the invitation from there, and you will be automatically added to the account.

New users are granted "Staff Access Level" by default as a security feature, which restricts access to all modules. To change the user's access level or grant him access to specific modules, go to Admin Module -> All Users. Select the user and carry on from there.

Need help adding users to your account? Contact our support team at support@netrink.com.
